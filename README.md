
```
usage: ./mmb <N> <mode> <test_count> <block_size> <outfile>

    N = matrix size
    mode = multiplication type -> 1 for A*A, 2 for A(t)*A, 3 for A*A(t), 4 for A(t)*A(t)
    test_count = number of test cases
    block_size = block size for block-to-block multiplication
    outfile = name for output file which will contain the data 
```
